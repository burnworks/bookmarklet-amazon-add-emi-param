# Add "emi" parameters

Amazon の URL に 「&emi=AN1VRQENFRJN5」 を足すとマーケットプレイスの出品を除外できるとのことなのでそれを簡単にするブックマークレット。

## 使い方

基本的な使い方は通常のブックマークレットと同じです。

    javascript:window.location.href+='&emi=AN1VRQENFRJN5';
